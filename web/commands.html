Wiki do TikTokWall.jar

Todos os comandos do plugin Minecraft.

Use esta página para entender o que cada comando faz, quando usar e qual exemplo copiar. No chat do Minecraft use a barra inicial. Via RCON, use o mesmo comando sem a barra.

Abrir painel

Como ler

Regras rápidas.

No jogo: digite com barra, por exemplo /tiktokwall test.

No painel/RCON: o portal envia sem barra, por exemplo tiktokwall test.

Permissão: o jogador precisa ser OP ou ter tiktokwall.admin.

Depois de mudar tamanho: rode /tiktokwall setup para recriar a parede no tamanho novo.

Operação

Preparar, testar e limpar.

/tiktokwall setup

Prepara automaticamente a área da live: limpa espaço, cria piso, cria parede, moldura, posição da parede, mundo sempre de dia e modo criativo.

Use quando instalou o plugin, mudou tamanho ou quer refazer a arena.
/tiktokwall setpos

Define manualmente o canto superior esquerdo da parede usando o bloco que o jogador está olhando.

Use se você construiu sua própria parede e não quer usar o setup automático.
/tiktokwall test

Renderiza uma imagem colorida de teste usando blocos.

Use para confirmar que o plugin está carregado e a parede está no lugar certo.
/tiktokwall clear

Limpa a área ativa da parede e remove o nome acima do painel.

Use quando quiser apagar uma imagem travada ou resetar antes de outro teste.
/tiktokwall info

Mostra mundo, origem, direção, tamanho, duração, dithering, animação, nome, fogos, HTTP e status ocupado/livre.

Use para diagnosticar configuração errada.

Parede

Tamanho e direção.

/tiktokwall size <32|48|64|128|256>

Define o tamanho padrão da parede em blocos.

O bot precisa usar o mesmo valor em AVATAR_SIZE. Para 256, a parede fica 256x256 blocos.
/tiktokwall facing <NORTH|SOUTH|EAST|WEST>

Define para qual direção a imagem vai crescer horizontalmente.

Use se a imagem estiver invertida, crescendo para o lado errado ou olhando para trás.

Visual

Dithering e animação.

/tiktokwall dithering <on|off>

Liga ou desliga a mistura de blocos para simular mais tons.

Para fotos de live, off costuma ficar mais limpo. on pode gerar ruído.
/tiktokwall ditheringstrength <0-64>

Controla a força do dithering.

Valores baixos são mais discretos. Valores altos criam mais pontilhado.
/tiktokwall animation <on|off>

Liga ou desliga animação padrão do plugin.

O bot também pode mandar animate:false por evento. Curtidas usam sem animação por padrão.
/tiktokwall animationspeed <1-32>

Define quantas linhas de blocos são renderizadas por tick durante a animação.

Número maior é mais rápido. Para 128 ou 256, use 8, 16 ou 32 se quiser transição rápida.

Nome e fogos

Texto acima da parede e efeitos.

/tiktokwall nameplate <on|off>

Liga ou desliga o nome do usuário acima do painel, desenhado com blocos.

Use on para lives. Use off se a parede estiver muito alta ou apertada.
/tiktokwall nameplatematerial <MATERIAL>

Escolhe o bloco usado no nome acima da parede.

Exemplos bons: BLACK_CONCRETE, WHITE_CONCRETE, YELLOW_CONCRETE.
/tiktokwall fireworks <off|like|gift|any>

Define quando o plugin solta fogos perto da parede.

gift é o recomendado: fogos só para rosa/gift. any solta em tudo.
/tiktokwall fireworkcount <1-20>

Define quantos fogos são lançados por evento.

Use 3 a 8 para não poluir a tela. Use 10+ só para momentos especiais.
/tiktokwall fireworkpower <0-3>

Define a força/altura dos fogos.

1 é o padrão. 0 é mais baixo e discreto.

API local

Endpoints HTTP do plugin.

GET /health

Retorna se o plugin está online, versão instalada e se a parede está ocupada.

O portal usa isso para mostrar versão instalada e status do plugin.
POST /render

Recebe avatar em PNG base64, converte pixels para blocos e renderiza na parede.

Campos principais: username, nickname, eventType, imageBase64, size, clearAfter, animate.
POST /test

Roda o teste colorido via HTTP, equivalente ao comando /tiktokwall test.

Útil para testar sem entrar no Minecraft, desde que o Paper esteja aberto.

Receitas

Combinações prontas.

/tiktokwall size 256 /tiktokwall setup /tiktokwall test
/tiktokwall dithering off /tiktokwall animation on /tiktokwall animationspeed 16 /tiktokwall nameplate on /tiktokwall fireworks gift
