Setup guiado

Instale sem adivinhar a ordem.

O Paper precisa criar `eula.txt`, `server.properties` e `plugins/` antes de instalar o TikTokWall. Esta página mantém a sequência certa e separa Windows/Linux.

Baixar pack completo Baixar só o plugin

Sistema

Escolha o ambiente.

Passo 1

Baixe o pack completo e extraia em uma pasta simples.

Use uma pasta como C:\TikTokMinecraftLive. Evite OneDrive, Desktop sincronizado ou pastas com acentos.

Dentro do ZIP

O que cada arquivo faz.

TikTokWall.jarPlugin pronto para copiar para `plugins/` do Paper.
start-interface-windows.batAbre a ponte local no Windows e instala dependências do bot.
start-interface-linux.shAbre a ponte local no Linux e instala dependências do bot.
scripts/windows/start-paper.batScript para iniciar o Paper no Windows.
scripts/linux/start-paper.shScript para iniciar o Paper no Linux.
bot/Bot Node.js que conecta TikTok Live, processa avatar e fala com o plugin.
docs/Guias completos para Windows e Linux.
web/updates.jsonHistórico de versões usado pela página de atualizações.
